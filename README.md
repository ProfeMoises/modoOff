# 🌍 Planeta Modo Off

**Planeta Modo Off** es una aplicación diseñada para fomentar el bienestar digital y la reconexión con el mundo real. A través de mecánicas de gamificación, motiva a los usuarios a reducir su tiempo frente a las pantallas y dedicar momentos de calidad a actividades analógicas, creativas y sociales.

---

## 📋 Especificación del Sistema

El sistema nace de la necesidad de combatir la fatiga digital y el uso excesivo de dispositivos móviles. Se basa en tres pilares fundamentales:

1.  **Enfoque y Desconexión:** Un temporizador que bloquea simbólicamente el uso del dispositivo para fomentar periodos de concentración o descanso.
2.  **Gamificación:** Un sistema de puntos, rachas y niveles que recompensa positivamente el tiempo "fuera de línea".
3.  **Acción Real:** Un catálogo de retos diseñados para ser realizados en el mundo físico (leer un libro, caminar, conversar cara a cara).

---

## 🚀 Tecnologías Usadas

La aplicación utiliza un stack moderno de desarrollo web para garantizar velocidad, seguridad y una experiencia de usuario fluida:

*   **Frontend:** [React 19](https://react.dev/) con [TypeScript](https://www.typescriptlang.org/) para un código robusto y tipado.
*   **Construcción:** [Vite](https://vitejs.dev/) para un entorno de desarrollo ultrarrápido.
*   **Estilos:** [Tailwind CSS 4](https://tailwindcss.com/) para un diseño responsivo y minimalista.
*   **Animaciones:** [Motion](https://motion.dev/) (anteriormente Framer Motion) para transiciones suaves y micro-interacciones.
*   **Backend & Base de Datos:** [Firebase](https://firebase.google.com/) (Firestore y Authentication) para el almacenamiento en la nube y sincronización entre dispositivos.
*   **Iconografía:** [Lucide React](https://lucide.dev/) para una interfaz visual clara y moderna.

## 📖 Manual de Usabilidad

### 1. Inicio y Registro
Al entrar, puedes usar la aplicación de forma anónima (los datos se guardarán en tu navegador). Para asegurar que no pierdas tu progreso, haz clic en **"Entrar"** e inicia sesión con tu cuenta de Google.

### 2. El Dashboard
Es tu centro de control. Aquí verás:
*   **Puntos Totales:** Tu moneda de cambio para logros.
*   **Racha Actual:** Cuántos días seguidos has usado la app.
*   **Minutos Off:** El tiempo total que has dedicado a desconectarte.

### 3. El Temporizador (Timer)
*   Selecciona un intervalo (15 min, 30 min, 1 hora o 2 horas).
*   Presiona **"Iniciar Modo Off"**.
*   Durante este tiempo, el objetivo es no usar el dispositivo. Al finalizar, recibirás puntos (2 puntos por cada minuto).

### 4. Retos (Challenges)
Explora la lista de retos categorizados (Salud, Creatividad, Social, Naturaleza). Cada reto completado te otorga puntos extra y suma a tus estadísticas globales.

### 5. Logros (Achievements)
A medida que acumulas puntos, desbloquearás insignias. Revisa esta sección para ver qué tan cerca estás de convertirte en un "Maestro de la Presencia".

## 🔮 Posibilidades de Mejora Potencial

El sistema está diseñado para ser escalable. Algunas mejoras futuras incluyen:

1.  **Funciones Sociales:**
    *   *Leaderboards:* Tablas de clasificación para competir amistosamente con amigos o la comunidad global.
    *   *Retos Compartidos:* Posibilidad de invitar a otros usuarios a desconectarse al mismo tiempo.
2.  **Análisis de Datos:**
    *   Gráficos detallados de progreso semanal y mensual.
    *   Integración con APIs de tiempo de pantalla para comparar el uso real vs. el tiempo en "Modo Off".
3.  **Expansión de Contenido:**
    *   Nuevas categorías de retos (ej. Meditación, Deporte, Cocina).
    *   Sistema de niveles con avatares personalizables que "evolucionan" al desconectarse.
4.  **Notificaciones Inteligentes:**
    *   Recordatorios basados en el uso del dispositivo para sugerir un descanso cuando se detecta fatiga digital.
5.  **Versión PWA / Móvil:**
    *   Mejorar la experiencia como Progressive Web App para que funcione como una app nativa con notificaciones push.
